# Fork of Uniswap V2

In-depth documentation on Uniswap V2 is available at [uniswap.org](https://uniswap.org/docs).

# Local Development

Currently we are using nodejs version 12. (`nvm install 12`).
Yarn is replaced by npm.

## Install Dependencies

`npm install`

## Compile Contracts

`npm run compile`

## Run Tests

`npm run test`
